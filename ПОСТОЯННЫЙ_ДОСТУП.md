# 📱 Как получить постоянный доступ к приложению на iPhone

## 🎯 Самый простой способ - EAS Update

### Шаг 1: Опубликуйте проект на серверах Expo

Откройте PowerShell/Terminal в папке `office-mobile` и выполните:

```bash
npx eas-cli update --branch production --message "Release for iPhone"
```

**Процесс займет 3-5 минут.** Дождитесь завершения.

---

### Шаг 2: Получите постоянную ссылку

После успешной публикации вы получите ссылку вида:
```
https://u.expo.dev/update/[update-id]
```

Или используйте готовую ссылку для вашего проекта:
```
exp://u.expo.dev/84b8be99-64c2-4a59-92de-1d52360aa577?channel-name=production
```

---

### Шаг 3: Добавьте в Expo Go на iPhone

#### Вариант A: Через QR-код
1. После публикации в терминале появится QR-код
2. Откройте Expo Go на iPhone
3. Нажмите "Scan QR Code"
4. Отсканируйте код
5. Приложение загрузится и сохранится в списке "Recently opened"

#### Вариант B: Через ссылку
1. Откройте Expo Go на iPhone
2. Нажмите "Enter URL manually"
3. Вставьте ссылку: `exp://u.expo.dev/84b8be99-64c2-4a59-92de-1d52360aa577?channel-name=production`
4. Приложение загрузится

---

### Шаг 4: Используйте приложение постоянно

✅ Теперь приложение:
- Доступно 24/7 без компьютера
- Находится в списке "Recently opened" в Expo Go
- Обновляется автоматически при каждой новой публикации

---

## 🔄 Как обновлять приложение

Каждый раз когда вы делаете изменения в коде:

```bash
cd Desktop/office-mobile
npx eas-cli update --branch production --message "Описание изменений"
```

Пользователи получат обновление при следующем запуске приложения!

---

## 🚀 Альтернатива: Development Build (для продвинутых)

Если вам нужно собственное приложение (не через Expo Go):

### 1. Установите на iPhone через EAS Build:
```bash
eas build --platform ios --profile preview
```

Это создаст `.ipa` файл, который можно установить через TestFlight или Ad-Hoc.

### 2. Требования:
- Apple Developer Account ($99/год)
- Настройка сертификатов

## ❓ Частые вопросы

**Q: Нужен ли мне платный аккаунт Expo?**
A: Нет, EAS Update бесплатен для проектов с ограниченным использованием.

**Q: Сколько раз могу публиковать обновления?**
A: Неограниченно на бесплатном плане.

**Q: Как удалить старые версии?**
A: Expo автоматически управляет версиями. Последняя версия всегда доступна.

**Q: Работает ли без интернета?**
A: Нет, Expo Go требует интернет. Для оффлайн работы нужен Development Build.
